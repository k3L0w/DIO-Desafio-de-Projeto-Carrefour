Desafio de Projeto
Reforce seu conhecimento em Git com um desafio totalmente prático, onde você executará todos os passos para a criação, atualiação e sincronização de um repositório no Github.
Com isso, poderá compartilhar anotações e exercícios no próprio repositório. Criando, dessa forma, o primeiro (de muitos) projetos de portfólio.

Requisitos:
Noções básicas de Git/Github:
- Introdução ao Git e ao Github (sugestão)
- Git devidamente instalado (Git Bash)
